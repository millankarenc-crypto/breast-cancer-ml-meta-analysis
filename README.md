# Breast Cancer Risk Prediction Meta-Analysis (R Code)

This repository contains the R scripts used for the systematic review and meta-analysis of machine learning models for breast cancer risk prediction.

## Software
- R version: X.X.X
- Packages: meta, metafor, dmetar

## Files
- main_analysis.R: Runs the full meta-analysis
- data_preprocessing.R: Cleans and prepares dataset
- meta_analysis.R: Performs random-effects meta-analysis

## Reproducibility
All scripts are provided to ensure transparency and reproducibility of results.
